# ADMSS Admin

Веб-приложение административной панели для системы **ADMSS** (управление дилерами, пользователями, лидами и вспомогательными сервисами). Клиент обращается к API продукта и предназначен для внутреннего или административного использования.

## Возможности

- **Авторизация** — вход по логину и паролю, защищённые разделы через приватный маршрутизатор.
- **Дилеры** — список и карточки дилеров (`/dashboard`).
- **Лиды** — просмотр и карточки лидов (`/dashboard/leads`, `/dashboard/lead/:id`).
- **Пользователи** — активные пользователи, удалённые, детальная карточка пользователя, настройки, права, экспорт данных, API-ключи.
- **Общие разделы** — импорт данных, шаблоны отчётов и печатных форм.
- **Микросервисы** — список и карточка сервиса (`/dashboard/microservices`).
- Отдельные пункты меню (например, биллинг и отчёты) могут вести на заглушку до подключения функционала.

## Технологии

- **React**, **TypeScript**
- **Create React App** (`react-scripts`) — сборка и dev-сервер
- **React Router v6** — маршрутизация
- **Axios** — HTTP-запросы
- **React Query** — кэширование и состояние запросов к API
- **Bootstrap 5**, **React Bootstrap**, тема **Metronic** (SCSS, готовые компоненты и стили)
- **Formik** + **Yup** — формы и валидация
- **Husky**, **lint-staged**, **ESLint**, **Prettier** — качество кода и pre-commit

## Требования

- **Node** (рекомендуется LTS, совместимый с зависимостями проекта)
- **npm** (или совместимый менеджер пакетов)

## Установка и запуск

```bash
npm install
npm start
```

Приложение откроется в режиме разработки (по умолчанию порт CRA, обычно `http://localhost:3000`).

Сборка production-артефактов:

```bash
npm run build
```

## API и конфигурация

Базовый API задаётся в `src/common/app-consts.ts` (константа `API_URL`, по умолчанию указывает на окружение **app.admss.com**). Для другого бэкенда измените эту константу или вынесите значение в переменные окружения по договорённости команды.

Учётные данные и сессия после входа хранятся в `localStorage` (см. константы `LOC_STORAGE_USER` и связанные в том же файле).
